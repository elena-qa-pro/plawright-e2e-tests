# Playwright E2E Tests

This project contains end-to-end UI tests built using Playwright with Page Object Model (POM).

---

## 🚀 Tech Stack
- Playwright
- JavaScript
- Node.js

---

## 📂 Project Structure

pages/
  login.page.js
  inventory.page.js

tests/
  saucedemo-login.spec.js

---

## ✅ Test Coverage

- Successful Login with valid credentials
- Error validation for invalid login
- Add product to cart abd verify cart badge

---

## 🌐 Tested Browsers

- Chromium
- Firefox
- WebKit

---

## ▶️ How to Run Tests

Install dependencies:

npm install

Run tests:

npx playwright test

Run tests in headed mode:

npx playwright test --headed

---

## 📊 Reports

To open the HTML report:

npx playwright show-report
